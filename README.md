# Vapi Manager Agent (Hackathon POC)

An autonomous Forensic Auditor for AI Voice Agents.

### The Workflow
1. The Analyzer: Extracts governance blueprints from Vapi System Prompts.
2. The Scorer: Audits call transcripts against blueprints for violations.
3. The Optimizer: Generates developer-ready prompt fixes and management reports.

### Tech Stack
- OpenClaw: Core agent framework (The Manager).
- Vapi: Voice Source.
- n8n: Action and Alerting layer.
